\subsection{Detailed Description}
C routine that generates the data for building the Figure 2(c). 

Figure 2(c) shows the curves of the reconstruction signal-\/to-\/noise ratio (R-\/\+S\+NR) in dB of the recovered coefficient vector versus the impulsiveness level of the noisy projections. For this experiment, the additive perturbations are characterized using an $ \varepsilon $-\/contaminated noise model (also known as p-\/\+Gaussian noise) with $ \sigma_2^2 = 100 \sigma_1^2 $. The variance of the Gaussian noise is computed in order to obtain a previously defined S\+NR value, i.\+e. $ SNR = 12 dB $. Furthermore, this experiment obtains the S\+NR values generated by both the proposed sparse signal recovery algorithm (W\+M\+R-\/\+AR, weighted median regressions with adaptive regularizer) and the method based on weighted median regressions with hard thresholding (W\+M\+R-\/\+HT). Each point in the curves are obtained by averaging a user-\/defined number of trials, which is introduced in running time. The results of the simulation are saved in the results directory and the curves can be visualized by running the {\ttfamily Figure2c.\+m} routine. This plotting routine can be executed under either Octave or Matlab.
